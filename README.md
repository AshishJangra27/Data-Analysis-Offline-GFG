## Description of Folders

### 1. Introduction to Excel
This folder contains introductory materials to get you started with Microsoft Excel. You'll learn about basic Excel functionalities, data entry, and simple formulas.

### 2. Conditionals & Null Values
Learn how to handle conditionals and null values in Excel to clean and prepare your data for analysis.

### 3. Data Cleaning with Excel
This section focuses on various data cleaning techniques in Excel, including removing duplicates, handling missing data, and more.

### 4. Myntra Data Cleaning
A practical project on data cleaning using a dataset from Myntra.

### 5. Dashboard Building with Excel
Learn how to build interactive dashboards in Excel to visualize your data effectively.

### 6. Databases 101
An introduction to databases, covering fundamental concepts and structures.

### 7. Getting started with SQL
Basic SQL concepts to help you get started with querying databases.

### 8. SQL with Datasets
Learn to work with datasets using SQL, including importing data and running queries.

### 9. Functions & Sorting
Understand how to use functions and sort data efficiently in SQL.

### 10. Groupby
Master the `GROUP BY` clause in SQL to aggregate data.

### 11. Joins
Learn about different types of joins in SQL to combine data from multiple tables.

### 12. ChatGPT & Cricket Players Analysis Assignment
A special project combining ChatGPT with cricket players' data analysis.

### 13. Operators & Subqueries
Dive into SQL operators and learn how to use subqueries for complex data retrieval.

### 14. Ranks
Understand how to rank data within your SQL queries.

### 15. Views & Data Cleaning
Learn to create views and perform data cleaning using SQL.

### 16. Getting started with Python
Introduction to Python programming for data analysis.

### 17. Conditionals (Python)
Learn how to use conditionals in Python.

### 18. Loops (Python)
Master looping constructs in Python to automate repetitive tasks.

### 19. Lists (Python)
Understand how to use lists in Python for data storage and manipulation.

### 20. Dictionaries (Python)
Learn about dictionaries in Python and their applications in data analysis.

### 21. Getting started with Pandas
Introduction to Pandas library for data manipulation and analysis in Python.

### 22. Exploratory Data Analysis
Techniques for exploring and understanding your data using Python.

### 23. Data Cleaning with Python
Learn various data cleaning techniques using Python.

### 24. Data Analysis from Scratch
A comprehensive guide to performing data analysis from scratch using Python.

### 25. Gaana Data Analysis
Project focused on analyzing data from Gaana, a music streaming service.

### 26. Data Analysis with Inbuilt Functions
Learn to use inbuilt functions in Python for data analysis.

### 29. Jobs Recommendation

## How to Use This Repository

1. **Clone the Repository**: 
   ```sh
   git clone https://github.com/AshishJangra27/Data-Analysis-Offline-GFG.git
   ```

2. **Navigate to the Folder of Interest**:
   ```sh
   cd Data-Analysis-Offline-GFG
   ```

3. **Open the Relevant Files**:
   - Each folder contains files and instructions relevant to the topic. Open these files in your preferred environment (e.g., Excel, Jupyter Notebook).

4. **Follow the Instructions**:
   - Each file or project will have detailed instructions to guide you through the learning process.
